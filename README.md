# ☕ Java POO - Exercícios

Repositório contendo exercícios resolvidos durante meus estudos de Java e Programação Orientada a Objetos (POO).

Os exercícios foram desenvolvidos seguindo uma progressão gradual de aprendizado, consolidando fundamentos de Java e Programação Orientada a Objetos por meio da implementação de regras de negócio, manipulação de coleções e boas práticas de desenvolvimento.

Os exercícios foram desenvolvidos com foco em:

- Programação Orientada a Objetos
- Estruturação e organização de projetos em Java
- Modelagem de entidades e regras de negócio
- Manipulação de coleções e estruturas de dados
- Tratamento de exceções e validação de dados
- Boas práticas de desenvolvimento

---

# 📚 Exercícios Disponíveis

### 1️⃣ Nota do Aluno

Sistema que calcula a nota final de um aluno e informa se ele foi aprovado ou reprovado.

### 2️⃣ Retângulo

Programa que calcula área, perímetro e diagonal de um retângulo.

### 3️⃣ Salário do Funcionário

Sistema que calcula o salário final de um funcionário com base em aumento percentual.

### 4️⃣ Conversor de Dólar

Programa que realiza a conversão de dólar para real incluindo taxa de IOF.

### 5️⃣ Controle de Produto

Sistema que realiza o cadastro de um produto com nome, preço e quantidade em estoque, permitindo adicionar e remover produtos e calcular o valor total em estoque.

### 6️⃣ Conta Bancária

Programa que realiza o cadastro de uma conta bancária permitindo depósito e saque, considerando depósito inicial opcional e taxa fixa aplicada em operações de saque.

### 7️⃣ Alturas

Programa que cadastra várias pessoas utilizando vetores, calcula a altura média e exibe a porcentagem e nomes das pessoas com menos de 16 anos.

### 8️⃣ Pensionato

Sistema que realiza o cadastro de aluguéis de quartos utilizando vetor de objetos e exibe os quartos ocupados.

### 9️⃣ Cadastro de Pessoas

Programa que realiza o cadastro de pessoas utilizando ArrayList e permite buscar uma pessoa pelo nome.

### 🔟 Produtos (Herança e Polimorfismo)

Sistema que cadastra diferentes tipos de produtos (comum, usado e importado) utilizando herança e polimorfismo, exibindo etiquetas de preço específicas para cada tipo.

### 1️⃣1️⃣ Imposto de Pessoas (Abstração)

Sistema que calcula impostos para pessoas físicas e jurídicas utilizando classes abstratas, herança e polimorfismo.

### 1️⃣2️⃣ Tratamento de Exceções

Sistema bancário que realiza saques aplicando regras de negócio através de exceções personalizadas e tratamento de erros.

### 1️⃣3️⃣ Alunos por Curso (Set)

Sistema que registra os alunos dos cursos A, B e C e calcula a quantidade total de alunos do instrutor, evitando duplicidades através da coleção `HashSet`.
