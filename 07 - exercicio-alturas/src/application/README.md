## Descrição

Programa que realiza o cadastro de várias pessoas informando nome, idade e altura utilizando vetores (arrays). Ao final, calcula a altura média das pessoas, a porcentagem de pessoas com menos de 16 anos e exibe os nomes dessas pessoas.

## Conceitos praticados

- Vetores (arrays)
- Estrutura de repetição (for)
- Estrutura condicional (if)
- Cálculo de média
- Cálculo de porcentagem
- Entrada de dados com Scanner
- Saída de dados formatada com printf

## Objetivo

Praticar o uso de vetores para armazenar múltiplos dados relacionados, realizar cálculos matemáticos e aplicar estruturas condicionais para análise de dados.
