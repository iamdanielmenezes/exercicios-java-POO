## Descrição

Programa que calcula o número total de alunos de um instrutor que possui três cursos (A, B e C). Como um mesmo aluno pode estar matriculado em mais de um curso, o sistema utiliza uma coleção do tipo Set para armazenar os alunos e evitar duplicidades.

Ao final, o programa exibe a quantidade total de alunos distintos.

## Conceitos praticados

- Interface Set
- HashSet
- Classes e objetos
- Encapsulamento
- Sobrescrita dos métodos equals e hashCode
- Estruturas de repetição (for)
- Entrada de dados com Scanner
- Eliminação de elementos duplicados em coleções

## Objetivo

Praticar o uso da interface Set para armazenar elementos únicos, compreendendo a importância da implementação dos métodos equals e hashCode para comparação de objetos.
