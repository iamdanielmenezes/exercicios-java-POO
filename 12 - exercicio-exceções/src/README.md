## Descrição

Programa que realiza operações de saque em uma conta bancária, aplicando regras de negócio para validação do valor solicitado.

O sistema utiliza exceções personalizadas para tratar situações como saldo insuficiente ou tentativa de saque acima do limite permitido, exibindo mensagens de erro adequadas ao usuário.

## Conceitos praticados

- Tratamento de exceções
- Exceções personalizadas
- Uso de throw
- Bloco try/catch
- Encapsulamento
- Regras de negócio
- Classes e objetos
- Entrada de dados com Scanner

## Objetivo

Praticar a criação e utilização de exceções personalizadas para controlar regras de negócio e tratar erros de forma adequada durante a execução do programa.
