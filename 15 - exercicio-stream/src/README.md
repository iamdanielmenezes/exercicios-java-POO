## Descrição

Programa que realiza a leitura de um arquivo contendo nome, e-mail e salário de funcionários.

Após carregar os dados, o sistema solicita um valor de salário e exibe, em ordem alfabética, os e-mails dos funcionários que recebem acima do valor informado. Em seguida, calcula a soma dos salários dos funcionários cujo nome inicia com a letra **'M'** utilizando a API de Streams.

### Exemplo de arquivo de entrada

```text
Maria,maria@gmail.com,3200.00
Alex,alex@gmail.com,1900.00
Marco,marco@gmail.com,1700.00
Bob,bob@gmail.com,3500.00
Anna,anna@gmail.com,2800.00
```

## Conceitos praticados

- API Stream
- Expressões Lambda
- Referência de métodos (`Method Reference`)
- Operações `filter`, `map`, `sorted` e `reduce`
- Comparator
- Manipulação de arquivos (`BufferedReader`)
- Programação funcional em Java
- Tratamento de exceções (`IOException`)

## Objetivo

Praticar o processamento de coleções utilizando a API de Streams, aplicando operações funcionais para filtragem, transformação, ordenação e redução de dados.
