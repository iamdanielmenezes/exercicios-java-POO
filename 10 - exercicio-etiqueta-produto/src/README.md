## Descrição

Programa que realiza o cadastro de diferentes tipos de produtos utilizando herança e polimorfismo. O sistema permite cadastrar produtos comuns, usados e importados, cada um com suas características específicas.

Após o cadastro, o programa exibe uma etiqueta de preço para cada produto, aplicando comportamentos diferentes de acordo com o tipo do objeto.

## Conceitos praticados

- Herança
- Polimorfismo
- Sobrescrita de métodos (Override)
- Uso da palavra-chave `super`
- Classes e objetos
- Lista de objetos (List)
- Estrutura condicional (if/else)
- Entrada de dados com Scanner
- Manipulação de datas (LocalDate)

## Objetivo

Praticar os conceitos de herança e polimorfismo, permitindo que diferentes tipos de objetos tenham comportamentos específicos através da sobrescrita de métodos.
