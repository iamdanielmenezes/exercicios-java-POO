## Descrição

Programa que realiza o cadastro de várias pessoas utilizando uma lista dinâmica (ArrayList), armazenando nome, idade e cidade. Após o cadastro, o sistema exibe a lista completa de pessoas e permite buscar uma pessoa pelo nome, informando se ela foi encontrada ou não.

## Conceitos praticados

- ArrayList
- Classes e objetos
- Estrutura de repetição (for-each)
- Estrutura condicional (if)
- Busca de dados em lista
- Uso de variável booleana (boolean)
- Encapsulamento (getters e setters)
- Método toString
- Entrada de dados com Scanner

## Objetivo

Praticar o uso de listas dinâmicas (ArrayList) para armazenamento de objetos, exibição de dados e implementação de busca simples dentro de uma coleção.
