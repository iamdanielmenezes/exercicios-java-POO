## Descrição

Programa que realiza o cadastro de pagadores de impostos, podendo ser pessoa física ou pessoa jurídica. Cada tipo de contribuinte possui regras específicas para cálculo de imposto.

Após o cadastro, o sistema exibe o valor de imposto pago por cada contribuinte e o total arrecadado.

## Conceitos praticados

- Classes abstratas
- Métodos abstratos
- Herança
- Polimorfismo
- Sobrescrita de métodos (Override)
- Lista de objetos (List)
- Estrutura condicional (if/else)
- Encapsulamento
- Entrada de dados com Scanner

## Objetivo

Praticar o uso de abstração através de classes e métodos abstratos, utilizando herança e polimorfismo para implementar diferentes regras de cálculo de imposto para cada tipo de contribuinte.
