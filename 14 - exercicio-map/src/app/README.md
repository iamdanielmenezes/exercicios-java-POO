## Descrição

Programa que realiza a leitura de um arquivo contendo o nome de candidatos e a quantidade de votos obtidos em diferentes urnas. Utilizando a interface `Map`, o sistema agrupa os votos de cada candidato e calcula a quantidade total de votos recebidos.

Ao final, o programa exibe a relação de candidatos e seus respectivos totais de votos em ordem alfabética.

### Exemplo de arquivo de entrada

```text
Alex Blue,15
Maria Green,22
Bob Brown,21
Alex Blue,30
Bob Brown,15
Maria Green,27
Maria Green,22
Bob Brown,25
Alex Blue,31
```

## Conceitos praticados

- Interface `Map`
- Implementação `TreeMap`
- Manipulação de arquivos (`BufferedReader`)
- Leitura e processamento de arquivos texto
- Estruturas condicionais
- Acumulação de valores utilizando Map
- Tratamento de exceções (`IOException`)

## Objetivo

Praticar o uso da interface `Map` para armazenar pares chave-valor, agrupando informações e acumulando valores de forma eficiente a partir da leitura de um arquivo.
