## Descrição

Programa que realiza o cadastro de aluguéis de quartos em um pensionato, armazenando nome e email dos hóspedes em posições específicas de um vetor. Ao final, exibe os quartos ocupados e seus respectivos hóspedes.

## Conceitos praticados

- Vetores de objetos
- Classes e objetos
- Estrutura de repetição (for)
- Estrutura condicional (if)
- Verificação de posições nulas (null)
- Entrada de dados com Scanner
- Método toString

## Objetivo

Praticar o uso de vetores com objetos, armazenamento de dados em posições específicas e verificação de posições ocupadas.
